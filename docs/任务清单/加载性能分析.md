## 加载性能分析

### 📊 核心数据

| 指标         | 数值                      |
| ------------ | ------------------------- |
| **加载数量** | 15,698,152 条（约1569万） |
| **加载时间** | 103.776 秒（约1分44秒）   |
| **加载速度** | **~151,271 条/秒**        |

### 🚀 性能评估

#### **当前性能**
- **每秒处理速度**：约 **15.1 万条/秒**
- 这个速度非常优秀！

#### **预计 8000 万条数据**
```
80,000,000 ÷ 151,271 ≈ 529 秒 ≈ 8.8 分钟
```

✅ **远超 30 分钟的目标！**

### ⚡ 性能优势分析

1. **simdjson 高性能解析**
   - GB/s 级别 JSON 解析速度
   - SIMD 指令并行处理

2. **批量处理优化**
   - 减少锁竞争（从数千万次减少到数千次）
   - 批量合并数据，开销更低

3. **并行处理架构**
   - 32 个解压线程
   - 32 个 JSON 处理线程
   - 流水线式处理（边解压边解析）

4. **数据结构优化**
   - 位压缩存储（CardInfo）
   - 布隆过滤器快速预检
