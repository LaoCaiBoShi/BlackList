我已经为您创建了一个完整的跨平台Qt网络通信程序，支持Windows和Linux系统。以下是项目的结构和功能：

### 项目结构
- **NetworkCommunicator.pro** - 项目配置文件，包含必要的Qt模块
- **main.cpp** - 程序入口点
- **mainwindow.h/cpp** - 主窗口类，处理UI和用户交互
- **server.h/cpp** - 服务器类，实现TCP服务器功能
- **client.h/cpp** - 客户端类，实现TCP客户端功能
- **mainwindow.ui** - 主窗口UI界面

### 功能特点
1. **服务器功能**：
   - 启动服务器并监听指定端口
   - 接收客户端连接
   - 接收和发送消息

2. **客户端功能**：
   - 连接到指定主机和端口
   - 发送和接收消息
   - 处理连接状态变化

3. **用户界面**：
   - 服务器控制区域（启动服务器按钮）
   - 客户端连接区域（主机、端口输入和连接按钮）
   - 消息发送区域（输入框和发送按钮）
   - 状态显示区域（显示连接状态和消息收发情况）

### 技术实现
- 使用Qt的**QTcpServer**和**QTcpSocket**类实现网络通信
- 采用信号槽机制处理网络事件
- 支持UTF-8编码的消息传输
- 跨平台设计，可在Windows和Linux系统上运行

### 如何运行
1. 使用Qt Creator打开项目文件 **NetworkCommunicator.pro**
2. 构建项目（Ctrl+B）
3. 运行程序（Ctrl+R）
4. 在界面中：
   - 点击"启动服务器"按钮启动服务器
   - 输入主机地址（默认127.0.0.1）和端口（默认8888），点击"连接服务器"按钮
   - 在消息输入框中输入消息，点击"发送"按钮发送消息
   - 在状态显示区域查看消息收发情况

### 测试方法
1. 在同一台机器上运行程序，启动服务器后连接本地主机
2. 也可以在不同机器之间测试，修改主机地址为服务器所在机器的IP地址

该项目提供了一个基础的网络通信框架，您可以根据需要扩展功能，如添加消息加密、文件传输等高级功能。
